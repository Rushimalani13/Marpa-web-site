4.1.9 General rules for the earleme variables

At this point, the most generalized input model has been introduced. Here are some useful facts about the earleme variables that will always be the case, no matter which of the input models is in use.

  • The current earleme is greater than or equal to the latest earleme.
  • The furthest earleme is greater than or equal to the latest earleme.
  • If the parser is not exhausted, the furthest earleme is always greater than or equal to the current earleme.
  • In an exhausted parser, the furthest earleme is always less than or equal to the current earleme.
  • If the furthest earleme is greater than the current earleme, the parser is not exhausted.
  • For the furthest earleme to be less than the current earleme, the parser must be exhausted.
