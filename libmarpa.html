About Libmarpa

Libmarpa is the C language library that implements the core of Marpa. It is part of Marpa::R2, a Perl CPAN module. For the moment, this entanglement makes sense -- all of Libmarpa's tests are in Perl, and you really don't want to install Libmarpa without testing it on your configuration.

Installing Libmarpa

Libmarpa's only requirements are Perl 5.10 and a C90 compiler. To install Libmarpa:

  • Download, install and test Marpa::R2. Please do not skip the testing.

  • The Libmarpa distribution is in the directory libmarpa_dist. The contents of this directory are unaltered, straight from a GNU autotools tarball. I went to some trouble to keep it very "vanilla", and it should support all the standard GNU autotools targets and install in all the standard ways.

Documentation

  • The Libmarpa API document is online in HTML and PDF versions. The HTML is also available in a one-page version.

  • Sources for the Libmarpa documentation are included in the Marpa::R2 distribution, in the directory libmarpa_doc_dist. Like the directory for the source, the contents of this directory are unaltered, straight from a "vanilla" GNU autotools tarball. It should support all the standard GNU autotools targets and install in all the standard ways.

Libmarpa and Perl

Libmarpa can run and even build on a system without Perl. It has no runtime, build-time or compile-time dependence on Perl. Testing, however, does require Perl and I strongly advise against running Libmarpa in a configuration on which it has not been tested.

Libmarpa internals

The source files for Libmarpa come as part of the Marpa::R2 distribution, and can also be found in the Marpa::R2 github repository. I worked hard to limit the prerequisites needed to configure, compile and run Libmarpa. Working on Libmarpa's internals is another matter. Libmarpa is literate programming -- and work on Libmarpa's internals requires, among other things, Knuth's CWeb and therefore Tex.

Libmarpa is highly mathematical code. Its internals are, frankly, daunting. For me, Marpa has a real beauty, but nobody could claim it achieves that beauty through simplicity. I once had hopes that reading Marpa would be like experiencing the effortless power of a late Picasso. But Angkor Wat and Hieronymous Bosch show that beauty comes in many forms.
